# TSS — Text-to-Speak Shosetu  
**小説家になろうの読み上げツール**

## ご利用にあたってのお願い

このソフトウェアはオープンソース（OSS）として公開されていますが、著作権は作者に帰属しています。  
そのため、**過度な改変や再配布はお控えください**。

もし改善や機能追加をご希望の場合は、[**プルリクエスト（Pull Request）**](https://github.com/TSS-extension/TSS---Text-to-Speak-Shosetu/wiki/Pull-Request) の形でご提案いただければ幸いです。

現時点（v1.2）ではサーバーとの通信機能は含まれていませんが、**v1.5 ではサーバーアクセス機能を実装予定**です。  
その際には、必要な情報を取得・利用する可能性がありますので、あらかじめご了承ください。
