# TSS — Text-to-Speak Shosetu  
**小説家になろうの読み上げツール**

## ご利用にあたってのお願い

このソフトウェアはオープンソース（OSS）として公開されていますが、著作権は作者に帰属しています。  
そのため、**過度な改変や再配布はお控えください**。

もし改善や機能追加をご希望の場合は、[**プルリクエスト（Pull Request）**](https://github.com/TSS-extension/TSS---Text-to-Speak-Shosetu/wiki/%E3%83%97%E3%83%AB%E3%83%AA%E3%82%AF%E3%82%A8%E3%82%B9%E3%83%88) の形でご提案いただければ幸いです。

現時点（v1.0）ではサーバーとの通信機能は含まれていませんが、**v1.1 ではサーバーアクセス機能を実装予定**です。  
その際には、必要な情報を取得・利用する可能性がありますので、あらかじめご了承ください。
